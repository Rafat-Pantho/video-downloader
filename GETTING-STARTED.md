## ⚡ Requirements

- Node.js (already installed ✓)

If yt-dlp is missing, the program will offer to install it automatically.

## 🔧 Manual yt-dlp Install

If auto-install doesn't work:

```bash
python -m pip install yt-dlp
```

Or:

```bash
pip install yt-dlp
```
